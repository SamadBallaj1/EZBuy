# User Stories

## User Roles

**Shopper (Guest)**: Anyone who visits the site to browse and purchase products without creating an account.

**Registered User**: A shopper who creates an account to access order history, faster checkout, and personalized features.

**Student**: A registered user with a verified .edu email who gets access to exclusive student discounts.

**Admin**: A site manager who can add, edit, or remove products from the inventory.

## User Stories

### Shopper Stories

1. As a shopper, I want to see all available products on the home page, so I can quickly browse what's in stock.

2. As a shopper, I want to click on a product to see more details, so I can learn about it before deciding to buy.

3. As a shopper, I want to add items to my cart, so I can keep shopping and buy everything at once.

4. As a shopper, I want to remove items from my cart, so I can change my mind if I decide not to buy something.

5. As a shopper, I want to see my cart total update in real-time, so I know how much I'm spending.

6. As a shopper, I want to checkout and pay securely with my card, so I can complete my purchase without worrying about safety.

7. As a shopper, I want to get a notification when I add something to my cart, so I know the action worked.

### Registered User Stories

8. As a registered user, I want to sign up and log in with my email or GitHub, so the site remembers me and saves my info.

9. As a registered user, I want to see all my past orders, so I can check what I bought and when.

10. As a registered user, I want my cart to stay saved even after I log out, so I don't lose what I was planning to buy.

11. As a registered user, I want faster checkout because my info is already saved, so I don't have to type everything again.

### Student Stories

12. As a student, I want to register using my .edu email, so I can prove I'm a student and get discounts.

13. As a student, I want to see discounted prices automatically on eligible products, so I don't have to look for coupon codes or do extra steps.

14. As a student, I want to see a "Student Discount" label on products, so I know which items are cheaper for me.

### Admin Stories

15. As an admin, I want to add new products to the store, so customers can see and buy the latest items.

16. As an admin, I want to edit product details like price or description, so I can fix mistakes or update info.

17. As an admin, I want to delete products that are discontinued, so the store only shows what's actually available.

### Stretch Stories (Optional Features)

18. As a shopper, I want to search for specific products, so I can find what I need faster instead of scrolling through everything.

19. As a shopper, I want to filter products by category or price range, so I can narrow down my options.

20. As a shopper, I want to sort products by price or newest arrivals, so I can browse in the order that makes sense for me.

21. As a registered user, I want to save products to a wishlist, so I can come back and buy them later.

22. As a registered user, I want to get personalized product recommendations, so I can discover items I might like.
